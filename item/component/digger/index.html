digger - 破壊ツール

説明: アイテムを破壊ツールにできる。

型: object

プロパティ

destroy_speeds

説明: 破壊速度とそのブロックの種類を決める。

型: object[]

アイテムプロパティ

block [必須]

説明: ブロックの種類。

型: object | string

プロパティ

name

説明: ブロックのID。

型: string

states

説明: ブロックのステート。

型: object

tags

説明: ブロックのタグ。molangで指定する。バニラブロックのタグ一覧
(例: query.any_tag('stone', 'wood')、query.all_tag('stone', 'minecraft:iron_tier_destructible') )

型: string

speed [必須]

説明: ブロックの破壊速度。

型: integer

use_efficiency

説明: エンチャントがこのアイテムでの採掘に影響するかどうか。trueでする。

型: boolean

例: 

"minecraft:digger": {
    "destroy_speeds": [
        {
            "block": {
                "tags": "query.any_tag('stone')"
            },
            "speed": 20
        }
    ],
    "use_efficiency": true
}

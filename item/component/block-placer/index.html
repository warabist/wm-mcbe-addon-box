block_placer - ブロック設置

説明: ブロックを設置できるようにする。

型: object

プロパティ

block [必須]

説明: 置くブロックの種類をIDで指定する。

型: string

例: 

"block": "minecraft:grass_block"

use_on

説明: 設置できるブロックのリスト。要素がstringのとき、それはブロックのID。

型: (object | string)[]

アイテムプロパティ

name [必須]

説明: ブロックのID。

型: string

例: 

"name": "minecraft:grass_block"

states

説明: ブロックのステート。

型: object

例: 

"states": {
    "height": 5
}

tags

説明: ブロックのタグ。

型: string

例: 

"tags": "stone"

例: 

"use_on": [
    {
        "name": "minecraft:barrel",
        "states": {
            "facing_direction": 1
        }
    },
    "minecraft:stone"
]

例: 

"minecraft:block_placer": {
    "block": "minecraft:grass_block",
    "use_on": [
        "minecraft:stone"
    ]
}

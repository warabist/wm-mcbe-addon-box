entity_placer - エンティティ設置

説明: エンティティを設置できるようにする。

型: object

プロパティ

dispense_on

説明: ディスペンサーでの、ブロックを設置できる条件。stringの場合はブロックのIDを入れる。

型: (object | string)[]

アイテムプロパティ

name

説明: ブロックのID。

型: string

states

説明: ブロックのステート。

型: object

tags

説明: ブロックのタグ。molangで指定する。バニラブロックのタグ一覧
(例: query.any_tag('stone', 'wood')、query.all_tag('stone', 'minecraft:iron_tier_destructible'))

型: string

entity

説明: 設置するエンティティのID。

型: string

use_on

説明: ブロックを設置できる条件。stringの場合はブロックのIDを入れる。

型: (object | string)[]

アイテムプロパティ

name

説明: ブロックのID。

型: string

states

説明: ブロックのステート。

型: object

tags

説明: ブロックのタグ。molangで指定する。バニラブロックのタグ一覧
(例: query.any_tag('stone', 'wood')、query.all_tag('stone', 'minecraft:iron_tier_destructible'))

型: string

例: 

"minecraft:entity_placer": {
    "entity": "minecraft:pig",
    "dispense_on": [
        {
            "tags": "query.any_tag('dirt')"
        }
    ],
    "use_on": [
        {
            "tags": "query.any_tag('dirt')"
        }
    ]
}

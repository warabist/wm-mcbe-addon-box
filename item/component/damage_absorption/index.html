damage_absorption - 吸収ダメージ設定

説明: アイテムが防具の時、このコンポーネントで指定したダメージの種類のみダメージが吸収される。吸収したダメージは耐久値を低下させる。wearable、durabilityコンポーネントが必要。

型: object

プロパティ

absorbable_causes [必須]

説明: 吸収させたいダメージの種類(minecraft damage causeで調べればでてくるはず)を指定する。

型: string[]

例: 

"minecraft:damage_absorption": {
    "absorbable_causes": [ "sonic_boom" ]
}

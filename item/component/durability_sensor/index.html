durability_sensor - 耐久センサー

説明: アイテムの耐久値が減ったときに指定したパーティクルやサウンド等のエフェクトを発生させることができる。

型: object

プロパティ

durability_thresholds [必須]

説明: 発生させるエフェクト、条件のリスト。条件を通るオブジェクトが複数ある場合、durabilityの値が低いオブジェクトが優先される。

型: object[]

アイテムプロパティ

durability

説明: アイテムの耐久値がこのプロパティで指定した値以下になれば、このエフェクトは発生する。

型: integer

particle_type

説明: 発生させるパーティクル名。

型: string

sound_event

説明: 再生するサウンド名。

型: string

例: 

"minecraft:durability_sensor": {
    "durability_thresholds": [
        {
            "durability": 1,
            "particle_type": "minecraft:huge_explosion_emitter"
            "sound_event": "random.explode"
        }
    ]
}

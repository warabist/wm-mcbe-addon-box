cooldown - クールダウン

説明: アイテムのクールダウンを設定する。

型: object

プロパティ

category [必須]

説明: クールダウンのカテゴリ。同じカテゴリのクールダウンを持つアイテムは全てクールダウンされる。カテゴリーはバニラのものだけでなく、自分で新しく作ることも可能。

型: string

例: 

"category": "wind_charge"

duration [必須]

説明: クールダウンの時間(秒単位)。

型: number

例: 

"duration": 1.5

例: 

"minecraft:cooldown": {
    "category": "my_food",
    "duration": 10
}
